# 人物头像图片目录

本目录用于存放历史人物的头像图片。

## 文件命名规则

- 文件名格式：`{人物ID}-{人物名称}.jpg` 或 `{人物ID}-{人物名称}.png`
- 例如：`1-大禹.jpg`、`8-秦始皇.jpg`

## 图片要求

- 格式：JPG 或 PNG
- 尺寸：建议 200x200 像素或更大（正方形）
- 大小：建议小于 100KB
- 内容：人物肖像、画像或雕塑照片

## 图片来源建议

1. **博物馆官网**：各大博物馆的文物图片
2. **历史资料网站**：如中国历史网、国学网等
3. **公开图片库**：如 Wikimedia Commons、百度百科等
4. **古籍插图**：古代人物画像、版画等

## 注意事项

- 确保图片版权清晰，建议使用公有领域或已授权的图片
- 图片应清晰可见，避免模糊或损坏
- 保持图片风格统一（建议使用古代画像风格）
